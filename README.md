# icqbot
Recipe bot for icqnew

# Init program

* Создаем виртуальное окружения для питона
```sh
python -m venv venv  
```
* Закрываем консоль на корзинку  
* Открываем консоль Ctrl+Shift+~  
* Устанавливаем сторонние библиотеки
```sh
pip install -r req.txt
```

# Парсить рецепты

```sh
python ./shef.py
```

# Запустить бота

```sh
python ./main.py
```
